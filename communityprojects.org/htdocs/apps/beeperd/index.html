Beeperd - Server Monitor
Beeperd runs in the background as a daemon monitoring an IP address and can take arbitrary actions based on the server being up/down. Default configuration is to call beeper with specified code when server no longer responds. Uses 'ping' and 'dip' right now, but this will be implemented through beeperd eventually, eliminating the need for external programs.
