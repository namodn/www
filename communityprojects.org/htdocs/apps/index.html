penemo
a network monitoring program written in perl. Implementing the basic features of snmp (mib2). The snmp features of penemo rely upon UCD-SNMP, though it can run and perform ping/http checks without it.
Developed and Maintained by Nick Jennings.

    Beeperd
    Performs specified function when the server cannot contact a given IP address ( default operation is to call a beeper ). Used for speedy notification when a server goes down. Now supports logfiles. Will soon have .conf to avoid unneccesary source hacks :)
    Please submit changes to robert@namodn.com via 'diff' files.
