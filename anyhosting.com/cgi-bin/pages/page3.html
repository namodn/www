FEATURES in protect.pl V 2.03

  • Ability to protect images with the add on script ImgProtect.pl.

  • Ability to protect frames pages.

  • Password protects as many .html files you like.

  • Ability to use subdirectories.

  • You can restrict access to different directories in the password list.

  • Password list with unlimited number of users and date of expiry.

  • Logging of all failed logins with info on IP address etc in a log file.

  • The login form can only be posted from your site.
